# Projeto PCMania

---

### **Resumo**
Este projeto foi desenvolvido para a disciplina de Programação Orientada a Objetos com o objetivo de simular um sistema simples de compras para a loja **PCMania**.

O sistema permite que o cliente escolha computadores de três promoções diferentes e, ao finalizar a compra, exibe:

`Dados do cliente`

`Informações completas dos computadores adquiridos`

`Valor total da compra`

`Mensagem de envio do pedido`

### **Uso de Inteligência Artificial**
Foi utilizada a ferramenta ChatGPT(modelo GPT-4) como apoio para:

**Organização do projeto em pacotes:**
* "Como eu separo corretamente as pastas (pacotes) desse projeto no IntelliJ pra cliente, computador, pedido e main?"
* "A classe Main tem que ficar dentro de um pacote obrigatoriamente ou posso deixar ela solta lá na raiz?"
* "Qual é a melhor forma de organizar as importações entre pacotes diferentes em Java?"

**Revisão estrutural e Correção do código**
* "Dá uma olhada nessas minhas classes e verifique se lógica dos for e dos vetores tá correta"
* "Leia as instruções do exercício em PDF e me diga se esqueci de alguma coisa."

**Estruturação correta das classes:**
* "Vê se eu coloquei os private nos lugares certos e se não criei getter e setter a mais."
* "Como eu crio o método static na classe ProcessarPedido pra ele receber o vetor de computadores?"
* "A parte de colocar o HardwareBasico e o SistemaOperacional dentro do Computador tá feita do jeito certo usando vetor normal?"

A IA foi utilizada apenas como ferramenta de apoio para revisão, tirar dúvidas e organização. Todo o entendimento, lógica e implementação final foram realizados individualmente.

Resultado do uso da IA: Satisfatório, auxiliando principalmente na conferência final da estrutura do projeto.
